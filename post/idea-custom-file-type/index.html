IntelliJ IDEA创建自定义文件类型模板

Step1

打开项目中想要制作模板的文件，如freemarker文件test.ftl，鼠标点击文件内任何位置（主要是让工具获取焦点，知道用户想以什么文件作为模板），然后点击菜单栏tool->Save File as Template...

Step2


打开如下图所示的窗口，窗口中会填写Name以及Extension，分别为右键新建菜单的名称（如：Freemarker File）和新建文件的后缀（如：ftl）。
在该模板内容中，可以使用IDEA给用户提供的常用信息变量以及自定义输入变量，如图中1表示自定义title的变量，当新建该类型文件时会提示用户输入title的值，图中2是系统提供的变量（${NAME}为文件名称），可以直接使用；系统提供的变量可以看窗口下半部分Description有解释说明，填好之后直接点保存即可

Step3

到这一步，我们就可以在项目中右键新建文件，选择我们在第二步保存的Name，如下图所示

Step4

打开新建文件的窗口，我们可以看到在窗口中除了要输入文件名意外，多了一个叫title的输入框，这就是我们在第二步中自定义的变量，输入保存后会直接在新建文件中输出

Step5

到这里就全部结束啦😁
