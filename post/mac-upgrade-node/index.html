macOS升级node版本

2019-05-23
macOS

Step1

安装 n 工具包，就一个字母“n”

sudo npm install -g n

Step2

好了，现在你已经完成了一大步，下面用 n 工具升级node到最新稳定版

sudo n stable

恭喜你现在node已经升级到最新稳定版，通过 node -v 查看当前版本，你也可以使用下面的命令来升级到指定版本

sudo n 8.9.0

在网上查资料看到说Mac使用 n 工具包升级node会出现问题，我这里没有出现😄
